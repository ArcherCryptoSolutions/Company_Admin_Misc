# ACA — Service Level Agreement (SLA)
Availability: Target >99% uptime.
Support: Documented incident response; internal ops channel.
Data Integrity: Timestamped, immutable snapshot logs and audit trails.
Change Management: Versioned deployments with rollback.
Security: Credential management, access controls, monitoring, and periodic reviews.
