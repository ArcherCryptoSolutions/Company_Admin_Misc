# ACA — Business plan
Archer Chain Analytics (ACA) is the internal audit/analytics agent for ACS subsidiaries. ACA provides multi-chain monitoring and integrity markers (e.g., Solana, ETH, BTC, Polygon, Avalanche, Cosmos, Near, XRP, XLM) with immutable logs and timestamped trails. Outputs feed CVR Essentials’ delayed records and ACS governance reporting.
