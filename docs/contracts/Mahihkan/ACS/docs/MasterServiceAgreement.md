# Master Service Agreement (MSA)
Parties: ACS (licensor) and subsidiaries (ACA, CVR Essentials, Mahihkan).
Scope: ACS licenses IP and provides umbrella governance; subsidiaries operate services under ACS oversight.
Term: Annual, renewable (12–36 months).
Fees: Inter-company licensing and resource transfers at industry-norm rates; invoiced monthly.
IP: Ownership remains with ACS; limited licenses to subsidiaries.
Confidentiality: Proprietary information protected; governed by service contracts.
Liability/Indemnity: Mutual indemnities; liability caps to be negotiated.
Dispute Resolution: Jurisdiction/forum to be set with counsel.
Compliance: AML/KYC, privacy, consumer protection, tax treatment to be reviewed with counsel.
